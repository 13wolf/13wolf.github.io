spring data entity update will set createdby createddate to null

2022-01-10 1 min read # audit # spring data # EnVers

当我们使用Spring Data并设置了Audit时，如果update数据时，Spring Data会将@Createdby @CreatedDate字段的数据设置为null，解决方法比较简单，直接在@Createdby @CreatedDate字段的@Column中设置updatable = false。

 @Column( updatable = false)
    @CreatedBy
    private String createdBy;

第二种方法是在Entity类上使用@DynamicUpdate 注解，参见@DynamicUpdate with Spring Data JPA

@Entity
@DynamicUpdate
public class Account {
    // Existing data and methods
}
